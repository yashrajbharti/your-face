# your-face

A minimalist, high-performance, full-screen camera Progressive Web App (PWA). No buttons, no clutter—just a clean, mirrored video feed of your face.

## ✨ Features
- **Ultra-Minimalist**: Full-window camera feed with zero UI clutter.
- **PWA Ready**: Installable on Android, iOS, and Desktop.
- **Offline Support**: Basic interface caching using Service Workers.
- **Privacy First**: No tracking, no external scripts, and no data collection.
- **Auto-Mirror**: Automatically flips the feed for a natural mirror experience.

## 🚀 Quick Start
1. Open `index.html` in any modern browser.
2. Grant camera permissions.
3. (Optional) Install as an app via your browser's "Add to Home Screen" or "Install" feature.

## 🛠 Tech Stack
- **Structure**: HTML5 Semantic Elements
- **Logic**: Vanilla JavaScript
- **PWA**: Web App Manifest & Service Workers
- **Styling**: Minimal CSS for full-viewport coverage
